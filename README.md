#  Developer Portfolio

A modern and fully responsive **Full Stack Developer Portfolio Website** built using **React.js**, **Vite**, and **Tailwind CSS**.  
This portfolio showcases my projects, coding profiles, skills, and contact information with smooth UI animations and dark/light theme support.

---

## 🌐 Live Demo

🔗 https://portfolio-new-ten-green.vercel.app/

---

## ✨ Features

- 🌙 Dark / Light Mode Toggle
- 📱 Fully Responsive Design
- 👨‍💻 Coding Profiles Section
  - LeetCode Stats
  - GitHub Activity
  - Contribution Streak
- 🛠️ Projects Showcase Section
- 📄 Resume Download Button
- 📬 Functional Contact Form using Web3Forms
- ⚡ Fast Performance with Vite
- 🎨 Modern UI with Tailwind CSS
- 🔥 Smooth Hover & Transition Effects

## 🧑‍💻 Getting Started (Local Setup)

Follow these steps to run the project locally:
```bash
# Clone the repository
git clone https://github.com/your-username/your-repo-name.git

# Navigate to project folder
cd your-repo-name

# Install dependencies
npm install

# Start development server
npm run dev
```

The app will run at:
```
http://localhost:5173
```

## 🔐 Environment Variables (Optional)

If you want to secure your EmailJS keys, create a `.env` file:
```env
VITE_WEB3FORMS_ACCESS_KEY=your_web3forms_access_key
```
